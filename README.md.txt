# Zomato Data Analysis

## Overview
This project performs Exploratory Data Analysis on Zomato dataset to understand restaurant trends and customer preferences.

## Tools Used
Python
Pandas
NumPy
Matplotlib

## Analysis Performed
- Data cleaning
- Missing value analysis
- Ratings distribution
- Country-wise analysis
- Cuisine popularity
- Cost distribution

## Outcome
Identified trends in ratings, cuisines, and pricing patterns.
